# telemetry-stream

FastAPI service for fan-out telemetry streaming.

## Key endpoints

- `GET /healthz`
- `GET /readyz`
- `GET /metrics`
- `GET /api/v1/streams/channels`
- `GET /api/v1/streams/stats`
- `GET /api/v1/streams/recent`
- `GET /api/v1/overlay/state`
- `WS /ws/telemetry`
- `WS /ws/overlay`
